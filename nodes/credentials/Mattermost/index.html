# Mattermost

You can find information about the operations supported by the Mattermost node on the integrations page. You can also browse the source code of the node on GitHub.

# Prerequisites

Create a Mattermost account.

# Using Access Token

  1. Access your System Console.
  2. Access Integrations.
  3. Go to Integration Management.
  4. Use the access token and Base URL with Mattermost node credentials in n8n.
